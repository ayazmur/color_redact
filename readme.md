# Red Shape Editor

Редактор для замены цветов в изображениях Word документов.

## Сборка

### Автоматическая (CI/CD)
При пуше в ветку `main` автоматически создается EXE файл в разделе Actions → Artifacts.

### Ручная сборка
```bash
# Установка зависимостей и сборка
build.bat

# Дополнительная оптимизация размера
python optimize.py
